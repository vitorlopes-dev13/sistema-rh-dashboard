# 🚀 Sistema de Gestão de Funcionários (RH)

Aplicação web interativa desenvolvida com Python e Streamlit para gerenciamento de funcionários com análise de dados em tempo real.

---

## 🌐 Acesse o projeto online

👉 [*(Acesso ao projeto)*](https://sistema-rh-dashboard-dbeurhm5grb6p6u4fuvw6z.streamlit.app/)

---

## 🎯 Sobre o Projeto

Este sistema foi desenvolvido com foco em prática de desenvolvimento e análise de dados, integrando:

* Manipulação de dados com Pandas
* Visualização com Plotly
* Banco de dados SQLite
* Interface interativa com Streamlit

---

## ⚙️ Funcionalidades

### 📊 Dashboard

* Indicadores (KPIs):

  * Total de funcionários
  * Salário médio
  * Maior salário
  * Folha salarial total
* Filtro de salários interativo
* Gráficos:

  * 📊 Barras
  * 📈 Histograma
  * 🥧 Pizza
* ⚠️ Detecção de outliers

---

### 👨‍💼 Gestão de Funcionários

* ➕ Cadastro
* 📋 Listagem
* 🔎 Busca por nome
* ✏️ Edição
* ❌ Exclusão

### 📁 Exportação

## ▶️ Como executar localmente

```bash
git clone https://github.com/vitorlopes-dev13/sistema-rh-dashboard.git
cd sistema-rh-dashboard
pip install -r requirements.txt
streamlit run dashboard.py
```
